# Control Framework Mapping

This document maps ERP integrity controls
to common governance frameworks.

## COSO Internal Control Framework

| COSO Component | ERP Implementation |
|---------------|-------------------|
| Control Environment | Defined ownership & escalation |
| Risk Assessment | Failure mode analysis |
| Control Activities | DB constraints & automation |
| Information & Communication | Incident & runbook documentation |
| Monitoring Activities | Scheduled integrity checks |
